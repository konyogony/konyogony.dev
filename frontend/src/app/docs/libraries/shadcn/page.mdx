# Shadcn

Shadcn is a modern UI library designed to provide beautiful and customizable components for React applications.

## Features

-   **Customizable:** Offers a range of components that are easy to customize to fit your design needs.
-   **Responsive Design:** Components are built to work well on all devices and screen sizes.
-   **Theming:** Supports theming to easily switch between different styles and color schemes.
-   **Accessibility:** Designed with accessibility in mind to ensure a better experience for all users.
